CSCI 4830 — Research Project

Predictive Event Modeling in Soccer Using Sequential Neural Networks

Traditional sports analytics relies on static statistics that fail to capture the fluid complexity of a game. This project leverages real-time positional data to move beyond historical summaries and enable sequence-aware event prediction.

Objectives

Real-Time Forecasting

The purpose of this project is to develop a machine learning model that predicts the probability of specific event outcomes, such as shots, crosses, or turnovers based on live positional data. While existing metrics evaluate past statistics, this research focuses on real-time forecasting to contribute to crucial play-making decisions.

The primary objective is to quantify how player movements, both on-ball and off-ball, influence the likelihood of in-game events. Coaches and analysts stand to benefit from these insights to maximize effectiveness and efficiency on the field.

Methodology

How It Works

This research utilizes large-scale event data collection, processing player coordinates and timestamps to build sequential model inputs. The core architecture is an LSTM (Long Short-Term Memory) network, chosen because soccer events are non-independent; the probability of a shot is highly dependent on the sequence of events preceding it.

Conclusion

Takeaways & Future Work

This project demonstrates that sequential, off-ball-aware models significantly outperform traditional static analytics. LSTM networks are well-suited for capturing the temporal dependencies inherent in soccer gameplay.

Implications: Coaches can use real-time probability dashboards to inform substitutions, pressing triggers, and set-piece strategies.

Future Work: Expand to other sports (basketball, hockey) with similar event structures; integrate video data alongside positional tracking; deploy as a live dashboard tool for club analysts.
